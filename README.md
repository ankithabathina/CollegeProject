# Student Management API

## 📌 Description
This is a Spring Boot REST API for managing student data.

## 🚀 Features
- CRUD Operations
- DTO Pattern
- Validation (@NotBlank, @Email)
- Exception Handling (@ControllerAdvice)
- REST APIs using Spring Boot

## 🛠 Technologies Used
- Java
- Spring Boot
- Spring Data JPA (Hibernate)
- MySQL

## 📂 Project Structure
- controller
- service
- repository
- entity
- dto
- exception

## ▶️ How to Run
1. Clone the repository
2. Open in Eclipse/IDE
3. Run as Spring Boot application
